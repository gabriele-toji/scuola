Introduzione a HTML

Cos'è HTML?

HTML (HyperText Markup Language) è il linguaggio standard per creare pagine web. Descrive la struttura di una pagina web utilizzando elementi e tag.

Struttura di base di un documento HTML

  • <!DOCTYPE html>: Dichiara il tipo di documento.
  • <html>: L'elemento radice del documento HTML.
  • <head>: Contiene i metadati e i collegamenti a stili o script.
  • <body>: Contiene il contenuto della pagina web.

Tag HTML comuni

  • <h1> a <h6>: Intestazioni
  • <p>: Paragrafo
  • <a>: Collegamento (link)
  • <img>: Immagini
  • <ul> e <ol>: Liste
  • <div> e <span>: Contenitori

Perché usare HTML?

HTML è essenziale per creare pagine web. È facile da imparare e consente di strutturare il contenuto in modo chiaro e leggibile. Inoltre, è il punto di partenza per imparare tecnologie web come CSS e JavaScript.
